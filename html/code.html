Working with Python

Python was used in an AWS Lambda function to download, clean, and upload data from a dataset file to AWS RDS. Next, we used Pandas and seaborn to analyze the data and provide insightful visualizations.

Working with MySQL

Used AWS Relational Database Service to spin up a MySQL server to store our data. Once a file in uploaded to our S3 bucket, a lambda function is triggered and the data is entered into our database for further analysis.
